The Project

Biology meets light.

Every model is rendered using a hand-written physically-based WebGL2 shading pipeline — metalness/roughness PBR, procedural image-based lighting, ACES filmic tone mapping, and PCF soft shadows. Zero external libraries.

01
Full PBR Shading Pipeline
Custom GLSL shaders implement Cook-Torrance BRDF with GGX normal distribution, Smith geometry term, and Fresnel-Schlick approximation — the same model used in film VFX pipelines.
02
Procedural Image-Based Lighting
A multi-sample hemispheric IBL probe is generated at runtime from gradient sky parameters, providing physically-plausible diffuse irradiance and specular radiance on all surfaces.
03
Zero External Dependencies
No Three.js, no CDN, no build step. Pure WebGL2 + vanilla JS. Works offline, in sandboxed iframes, and on GitHub Pages without any configuration.
04
Touch-Native Interaction
Multi-touch orbit, pinch-to-zoom, and momentum damping tuned for iPhone 16 ProMotion at 120Hz.
